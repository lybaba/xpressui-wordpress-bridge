# WordPress.org Assets

Ces fichiers sont uploadés via SVN dans le répertoire `/assets/` du plugin WordPress.org.
Ils ne sont PAS inclus dans le ZIP du plugin.

## Prêts

- `icon.svg` — logo IAKPress (SVG, réutilisé de iakpress)
- `icon-128x128.png` — logo IAKPress 128px
- `icon-256x256.png` — logo IAKPress 256px

## À créer

- `banner-772x250.png` — bannière page plugin (772×250 px)
- `banner-1544x500.png` — bannière retina (1544×500 px)
- `screenshot-1.png` — liste des soumissions wp-admin
- `screenshot-2.png` — détail d'une soumission
- `screenshot-3.png` — page Manage Workflows
- `screenshot-4.png` — formulaire inline dans une page WordPress
